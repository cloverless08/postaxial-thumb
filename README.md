# Project Overview

An assistive technology that grants the user a mechanized and opposable thumb controlled by another digit on the user's hand.

Problem Statement:
**"Workers with disabilities that affect their grip control, either due to poor strength, motor control, or missing appendages need an augmented assistive technology to aid ddeficits in grip ability"**

# General Information

This is a EGG1040 Engineerig Projects class project, we were given a budget of $200 to prototype, test and build an assistive technology. We publicize our work in the hope that it serves to aide or inspire any other projects out there.
